제어 구조 — 선택 (if)

명령어의 실행 순서를 제어하는 구조로 순차 · 선택 · 반복 세 가지가 있습니다.

📋 순차 구조
위에서 아래로 순서대로 실행. 프로그램의 가장 기본 구조입니다.
🔀 선택 구조
조건이 참/거짓에 따라 다른 명령을 실행. if, if~else, if~elif~else
🔁 반복 구조
조건을 만족하는 동안 명령을 반복 실행. for, while
① if 문
if 문 기본 구조
if 조건:
    # 조건이 참일 때 실행 (들여쓰기 필수!)
    print("조건이 참입니다.")
② if ~ else 문
학번_p133_2.py — 짝수/홀수 판별
number = int(input('숫자를 입력하세요: '))
if number % 2 == 0:
    print(number, '는 짝수입니다.')
else:
    print(number, '는 홀수입니다.')
③ if ~ elif ~ else 문
학번_p134.py — 양수/음수/0 판별
number = int(input('숫자를 입력하세요: '))
if number > 0:
    print('양수', end='')
elif number < 0:
    print('음수', end='')
else:

④ 중첩 제어 구조
✅
중첩 제어 구조는 조건문 안에 조건문, 반복문 안에 조건문 등 제어 구조 내부에 다른 제어 구조가 포함된 형태입니다. 들여쓰기로 계층을 표현합니다.

05 🔁

반복 구조 — for / while

① for 문
💡
for 문은 주어진 반복 범위만큼 명령어를 반복합니다. 반복 범위에는 리스트, 튜플, 문자열, range() 함수 등이 올 수 있습니다.
리스트 활용
fruits = ['사과', '바나나', '포도']
for fruit in fruits:
    print(fruit)
range() 활용
# 0~4 출력
for i in range(5):
    print(i)

# 1~9 홀수 출력
for i in range(1, 10, 2):
    print(i)
📌
range(시작, 끝, 간격): 시작 기본값 0, 끝-1까지의 정수 범위를 만듭니다. range(5) → 0, 1, 2, 3, 4
학번_99dan.py — 구구단 (중첩 for)
for dan in range(2, 10):
    print(f"--- {dan}단 ---")
    for i in range(1, 10):
        print(f"{dan} x {i} = {dan * i}")
    print()
② while 문
학번_p140.py — 숫자 업다운 게임
import random
count = 0
guess = 0
answer = random.randint(1, 100)

while answer != guess:
    guess = int(input('숫자는 무엇일까요?: '))
    count += 1
    if guess > answer:
        print('정답보다 크다.')
    elif guess < answer:
        print('정답보다 작다.')

print(f'정답! {count}번 만에 맞혔습니다.')
③ break & continue

🛑 break

반복문을 완전히 종료하고 다음 명령으로 이동합니다. 보통 if문과 함께 사용합니다.

⏭️ continue

현재 반복만 건너뛰고 다시 반복문으로 돌아가 이어서 실행합니다.

break vs continue 비교
score = [10, 20, -10, 30]

# break: 음수를 만나면 즉시 종료
for i in score:
    if i < 0: break
    print(i)   # 10, 20 출력

# continue: 음수만 건너뜀
for i in score:
    if i < 0: continue
    print(i)   # 10, 20, 30 출력
④ 무한 루프 (while True)
학번_p142.py — 곱셈 문제 게임
import random
while True:
    a = random.randint(1, 9)
    b = random.randint(1, 9)
    print(a, '*', b, '= ?')
    num = int(input('정답 입력: '))
    if num == a * b:
        print('정답입니다!')
        break
    print(f'틀렸습니다. 정답은 {a * b}입니다.')

📬 외장 함수

✍️ 사용자 정의 함수

직접 def로 정의해서 사용하는 함수. 코드 재사용성과 유지보수성 향상!

함수 정의 & 호출
함수 기본 구조
def 함수이름(매개변수):
    # 실행할 코드
    return 반환값

# ─── 예제 1: 인사 함수 ───
def greet(name):
    print("안녕하세요,", name, "님!")

greet("지우")   # → 안녕하세요, 지우 님!

# ─── 예제 2: 덧셈 함수 ───
def add(a, b):
    return a + b

result = add(5, 7)
print("결과:", result)   # → 결과: 12
지역변수 vs 전역변수

🟡 지역변수 (Local)

함수 내부에서 선언된 변수. 함수 안에서만 사용 가능하며, 함수 종료 시 메모리에서 제거됩니다.

🔵 전역변수 (Global)

함수 외부에서 선언된 변수. 모든 영역에서 사용 가능. 함수 안에서 값을 변경하려면 global 키워드 필요.

global 키워드 사용 예
count = 0    # 전역변수

def increase():
    global count   # 전역변수 사용 선언
    count += 1

increase()
print(count)    # → 1
🎮 함수 활용 — 숫자 맞히기 게임
숫자 맞히기 게임 (함수 활용)
import random

def 게임():
    정답 = random.randint(1, 100)

다차원 데이터 — 리스트 & 딕셔너리

📊
여러 개의 값이 모이면 1차원 데이터, 여러 개의 1차원 데이터가 모이면 2차원 데이터가 됩니다. 행(row)과 열(column)로 표현합니다.
① 1차원 리스트
리스트 기본 조작
T_size = [95, 85, 100, 95, 90]

# 인덱스로 접근 (0부터 시작)
print(T_size[0])    # → 95 (1번 학생)
print(T_size[1])    # → 85 (2번 학생)

# 값 수정
T_size[0] = 100

# 값 추가
T_size.append(95)
② 1차원 딕셔너리
딕셔너리 기본 조작
T_size = {'내빈':95, '미송':85, '민수':100}

# 키로 값 가져오기
print(T_size['내빈'])     # → 95

# 값 수정
T_size['내빈'] = 100

# 새 키:값 추가
T_size['재우'] = 105

# 딕셔너리 순회
for menu, price in T_size.items():
    print(f"{menu}: {price}")
③ 2차원 리스트
2차원 리스트 생성
# 6×6 크기의 0으로 채워진 2차원 리스트
maps = [[0 for i in range(6)] for i in range(6)]

# 행[row], 열[col]로 접근
maps[2][0] = 1   # 3행 1열에 값 변경

# 2중 for 문으로 전체 출력
for i in range(len(maps)):
    for j in range(len(maps[i])):
        print('★' if maps[i][j] else '□', end='')
    print()

08 🏗️

객체와 클래스 — 객체지향 프로그래밍

🏛️
객체지향 프로그래밍(OOP)은 객체(속성+메소드)를 기본 단위로 프로그램을 설계하는 방식입니다. 클래스는 객체를 만드는 설계도이고, 인스턴스는 메모리에 실체화된 객체입니다.

🧩 객체 (Object)

속성(상태 데이터) + 메소드(동작 함수)를 하나로 묶은 단위. 예: 자동차 객체 → 색상(속성), 전진(메소드)

📐 클래스 (Class)

동일한 종류의 객체를 여러 개 만들기 위한 설계도. 클래스 이름은 일반적으로 대문자로 시작합니다.

⚡ 인스턴스 (Instance)

클래스로부터 생성된 실체화된 객체. 동일 클래스에서 여러 인스턴스를 만들 수 있습니다.

클래스 정의 & 생성자
Book 클래스 — 생성자 활용
class Book:
    def __init__(self, title, author, year):
        # __init__ : 객체 생성 시 자동 호출되는 생성자
        self.title  = title
        self.author = author
        self.year   = year

# 객체(인스턴스) 생성
book1 = Book('톰소여의 모험', '마크 트웨인', 1876)
book2 = Book('데미안',       '헤르만 헤세',  1919)

print(book1.title, book1.author, book1.year)
print(book2.title, book2.author, book2.year)

톰소여의 모험 마크 트웨인 1876
데미안 헤르만 헤세 1919
클래스 완성 예제 — 도서 대여 시스템
Book & Student 클래스
class Book:
    def __init__(self, title, author):
        self.title     = title
        self.author    = author
        self.is_rented = False

class Student:
    def __init__(self, id, name):
        self.id           = id
        self.name         = name
        self.rented_books = []

    def rent_book(self, book):
        if not book.is_rented:
            book.is_rented = True
            self.rented_books.append(book)
            return f'{self.name}에게 {book.title} 대여함.'
        else:
            return f'{book.title} 대여 중.'

s2 = Student(2, '이순신')
print(s2.rent_book(b1))   # → 소나기 대여 중.

핵심 용어 정리
• 클래스(Class): 객체를 만드는 설계도
• 인스턴스(Instance): 메모리에 실체화된 객체
• 메소드(Method): 클래스 안에 정의된 함수
• 생성자(__init__): 객체 생성 시 자동으로 호출되는 메소드
• self: 인스턴스 자기 자신을 가리키는 참조
